IoT Smart Blind Stick

An assistive device for the visually impaired using sensors, GPS, and GSM modules.

Overview

This Smart Blind Stick is an IoT-based assistive device designed to help visually impaired individuals navigate safely. It uses multiple sensors and modules to detect obstacles, pits, and even provide voice guidance and location tracking.

Key Features:
  • Obstacle and pit detection using ultrasonic sensors
  • Real-time location tracking with GPS
  • Emergency SMS alerts through GSM module
  • Voice feedback for navigation
  • Bluetooth connectivity for additional commands

Technologies & Components

ESP32

Main microcontroller for logic & control

Ultrasonic Sensor

Detects nearby obstacles and pits

GSM 800L

Sends SMS alerts with GPS data

Bluetooth (HC-05)

Connects to smartphone for commands

Project Outcomes

  • Successfully demonstrated real-time obstacle and pit detection
  • Integrated GPS + GSM for emergency communication
  • Voice feedback system helped improve user interaction
  • Tested in real-world navigation scenarios with positive feedback
